## ROI 分析

| 投入 | 产出 | ROI |
|------|------|-----|
| explain 命令 (~40min) | 9 个命令的深度教学解释 | ⭐⭐⭐⭐⭐ 极高 — 核心教学功能 |
| tutorial (~10min) | 首次体验引导 | ⭐⭐⭐⭐ 高 |
| /proc 扩展 (~15min) | 4 个新 cat 可读条目 | ⭐⭐⭐⭐ 高 — 低成本高可见 |
| interrupts (~10min) | 中断机制可观测 | ⭐⭐⭐ 中 |
| boot 优化 (~5min) | 减少噪音输出 | ⭐⭐ 低成本微改善 |

## 关键学习

1. **explain 是最高 ROI 的教学功能** — 用纯文本就能传递复杂概念，不需要任何运行时机制
2. **/proc 扩展是"免费"功能** — 每个新条目只需一个格式化函数，但用户可以通过 `cat` 以一致的方式访问所有内核状态
3. **boot 优化的最大收益是移除串口日志** — 串口输出是 BIOS 引导的主要瓶颈

## 关键学习

1. **klog! 在 host-target 测试中会 SIGSEGV** — 因为调用了 serial 硬件端口。解决方案: `#[cfg(target_os = "none")]` 包裹硬件访问
2. **日志 ring buffer 不能和 trace ring buffer 共用** — 职责不同: 日志是文本消息，trace 是结构化 span，查询方式和存储粒度完全不同
3. **AtomicU8 做日志级别过滤比 Mutex 好** — 热路径（每次 klog 调用）只需一次原子读，避免锁开销

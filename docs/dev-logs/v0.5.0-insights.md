## 关键学习

1. **crash 实验不能真正 panic** — OOM 模拟必须在 90% 水位停止，否则后续操作会 panic
2. **脚本执行必须释放 VFS 锁** — `cmd_run` 在解析文件内容后必须 `drop(vfs_guard)` 再执行命令，否则命令中的 FS 操作会死锁
3. **Shell host 测试受限于 `#[global_allocator]`** — `LockedHeap::empty()` 在 host 环境未初始化，导致 shell crate 的 host-target 测试 SIGABRT

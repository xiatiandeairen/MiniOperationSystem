## 关键学习

1. **ISR 中绝不能获取 Mutex** — timer_tick 中的 trace_event! 与主线程的 trace_span! 竞争同一个 TRACER.buffer Mutex，导致死锁。这是 v0.2.0 唯一的 bug，但影响是致命的（键盘完全无响应）
2. **no_std 环境没有 str::parse** — 需要手写 parse_u32 和 parse_hex
3. **教学注解的 ROI 很高** — `describe_span()` 用极少代码让 trace 输出从数据变成知识

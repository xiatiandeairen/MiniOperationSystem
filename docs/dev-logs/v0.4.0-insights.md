# v0.4.0 开发 Insights

## 代码变更

| 指标 | v0.3.0 | v0.4.0 | 变化 |
|------|--------|--------|------|
| 代码行数 | ~7,500 | ~8,200 | +700 |
| 源文件 | 70 | 73 | +3 (compare.rs, lab.rs, env_cmds.rs) |
| Shell 命令 | 27 | 33 | +6 |
| compare 主题 | 0 | 5 | +5 |
| lab 实验 | 0 | 5 | +5 |

## ROI 分析

| 投入 | 产出 | ROI |
|------|------|-----|
| compare 命令 (~30min) | 5 主题深度对比 + "为什么" 解释 | ⭐⭐⭐⭐⭐ 核心教学差异化 |
| lab 命令 (~40min) | 5 个可执行实验 + 思考题 | ⭐⭐⭐⭐⭐ 最深度学习方式 |
| history + !! (~15min) | Shell 日常使用效率 | ⭐⭐⭐⭐ 高 (低成本高体验) |
| signal (~10min) | 进程生命周期完整性 | ⭐⭐⭐ 中 |
| env vars (~15min) | "真实 OS" 感 | ⭐⭐ 低 |

## 关键学习

1. **compare 的文本内容是最高 ROI 的教学资产** — 每个对比主题 ~20 行 println，但传递的设计洞察比代码更有价值
2. **lab 实验不需要复杂的验证框架** — 直接调用已有 API + 打印结果 + 思考题就是一个完整的实验
3. **Shell history 用固定数组而非 Vec** — 避免 alloc 依赖，ring buffer 模式在 no_std 中是万能解法

## 关键学习

1. **管道实现用输出捕获而非进程间通信** — 在单线程协作式 OS 中，管道是"先执行左侧捕获输出，再喂给右侧"的顺序操作，不需要并发
2. **AtomicU32 bitmap 是最轻量的进度追踪** — 32 bit 追踪 17 步，无需堆分配，无需锁
3. **在命令 handler 末尾加 mark() 是非侵入式的** — 不改变命令行为，只记录"用过"

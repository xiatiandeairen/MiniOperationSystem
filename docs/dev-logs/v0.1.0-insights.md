## 关键学习

1. **帧缓冲是用户体验的关键路径** — bootloader_api v0.11 使用像素帧缓冲而非 VGA text mode，这是项目最大的技术意外
2. **上下文切换不能在 kernel_main 线程中抢占 Shell** — 需要将 Shell 注册为独立任务才能安全抢占
3. **Trace Mutex 在 ISR 中会死锁** — timer 回调不能调用任何可能锁 Mutex 的函数
4. **BGR 不是 RGB** — framebuffer 像素格式是 BGR，颜色字节顺序反转
5. **串口 read_byte + without_interrupts** — 串口读取必须禁中断，否则与 serial_println 竞争

# MiniOS v0.6.0 — "Connect the Dots"

**发布日期**: 2026-02-26

## 版本概览

v0.6.0 将 38 个离散命令串联成完整的学习体验。管道让命令可以组合，journey 把命令编排成 4 章课程，graduation 给出结业总结，trace 数据可以端到端流入浏览器可视化。

## 新增特性

### 管道操作符 `|`
```
MiniOS $ cat /proc/meminfo | grep Free
MemFree:   63745 frames (254980 KiB)
MiniOS $ ps | head 2
PID  STATE    PRIORITY  CPU_TIME
0    READY    3         5000
```

### journey — 4 章节学习路径
```
MiniOS $ journey
Chapter 1: First Steps         [████░░░░░░] 40%
  ✅ help — see all commands
  ✅ ps — view processes
  ⬜ meminfo — check memory
  ...
Next step: Try 'meminfo' to check memory!
```

### graduation — 学习完成报告
```
MiniOS $ graduation
🎓 Congratulations! You completed the MiniOS learning journey!
OS concepts you explored:
  ✅ Process management, Memory management, Filesystem...
```

### alias — 命令别名
```
MiniOS $ alias ll ls /
MiniOS $ ll
  d      0  dev  ...
```

### trace 端到端数据通路
- `trace export` 输出带 `---TRACE-BEGIN/END---` 标记的 JSON
- `scripts/capture-trace.sh` 自动捕获并提取
- trace-viewer 可加载真实内核数据

# MiniOS v0.7.0 — "See Every Step"

**发布日期**: 2026-02-26

## 版本概览

v0.7.0 建立了完整的内核日志体系。学习者可以通过 `log level debug` 开启详细日志，用 `log module memory` 过滤只看内存子系统，用 `trace filter syscall` 只看系统调用链路。7 个子系统全部埋入 klog! 探针。

## 新增特性

### 内核日志系统 klog!
5 级日志（ERROR/WARN/INFO/DEBUG/TRACE），运行时可调：
```
MiniOS $ log level debug
MiniOS $ log module memory
MiniOS $ log history 10
[DEBUG] [memory] frame allocated: #16234
[DEBUG] [memory] frame freed: #16234
```

### trace filter — 模块级 span 过滤
```
MiniOS $ trace filter syscall
MiniOS $ trace list
(only showing syscall module spans)
MiniOS $ trace follow ls / --filter fs
(only showing fs module spans)
```

### 7 子系统日志埋点
| 子系统 | 埋点数 | 级别 | 示例 |
|--------|--------|------|------|
| memory | 3 | DEBUG+INFO | frame alloc/free, heap init |
| fs | 3 | DEBUG+TRACE | open/read/write 参数 |
| scheduler | 2 | DEBUG+INFO | preempt/boost |
| process | 2 | INFO | create/exit |
| syscall | 1 | DEBUG | dispatch args |
| ipc | 1 | DEBUG | send details |

### debug/log/explain 命令
- `debug on/off` — 一键切换详细模式
- `log level/module/history/off` — 精细控制
- `explain log` — 教学解释日志系统

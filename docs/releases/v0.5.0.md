# MiniOS v0.5.0 — "Break Things to Learn"

**发布日期**: 2026-02-26

## 版本概览

v0.5.0 引入故障实验室、脚本执行和文本处理工具，让用户从"安全操作"进阶到"破坏→理解→自动化"。

## 新增特性

### crash 命令 — 故障实验室
5 个安全的内核故障实验，每个带详细解释和自动恢复：
- `crash oom` — 耗尽堆内存，对比 MiniOS panic vs Linux OOM killer
- `crash stack` — 模拟栈溢出，解释 Page Fault → Double Fault → Triple Fault 链
- `crash divide-zero` — 解释 CPU 除零异常的处理路径
- `crash null-deref` — 解释空指针保护页和 SIGSEGV
- `crash fork-bomb` — 快速创建进程直到表满，对比 Linux ulimit/cgroups

### run 命令 — 脚本执行
```
MiniOS $ run /etc/init.sh
> echo MiniOS initialized successfully.
MiniOS initialized successfully.
```
支持 `#` 注释行，空行跳过。

### /etc/init.sh — 启动脚本
Shell 启动时自动执行，可自定义初始化流程。

### 文本处理工具
- `head [N] <file>` — 显示前 N 行（默认 10）
- `grep <pattern> <file>` — 过滤包含模式的行
- `wc <file>` — 统计行数/字数/字节数

### 测试覆盖提升
- Shell parser 测试（6 tests）
- contains_pattern 测试（5 tests）

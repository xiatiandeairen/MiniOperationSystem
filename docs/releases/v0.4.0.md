# MiniOS v0.4.0 — "Feel the Difference"

**发布日期**: 2026-02-26

## 版本概览

v0.4.0 引入对比学习、动态实验和 Shell 体验增强，让用户从"看见 OS 内部"进阶到"感受设计决策背后的 why"。

## 新增特性

### compare 命令 — 对比学习
5 个主题的 MiniOS vs Linux 设计对比表 + 深度解释：
- `compare scheduler` — MLFQ vs CFS，为什么 Linux 不用 MLFQ
- `compare memory` — Bitmap vs Buddy, 为什么 Linux 用 Slab
- `compare filesystem` — RamFS vs ext4, 为什么需要日志
- `compare ipc` — 消息队列 vs pipe/socket/shm 完整生态
- `compare syscall` — 函数调用 vs int 0x80, 为什么需要特权级

### lab 命令 — 5 个动态实验
| Lab | 主题 | 学到什么 |
|-----|------|---------|
| scheduler-fairness | CPU 时间分配 | MLFQ 优先级如何影响调度 |
| memory-usage | 堆分配/释放 | 分配器如何跟踪内存 |
| page-table-walk | 地址翻译 | 4级页表的每一步 |
| trace-overhead | trace 性能 | 观测系统的代价 |
| fs-operations | 文件生命周期 | create→write→read→delete 完整流程 |

### Shell 体验增强
- `history` — 查看命令历史 (32 条)
- `!!` — 重复上一条命令
- `signal <pid> <stop|continue|kill>` — 进程信号
- `set <key> <value>` / `env` — 环境变量

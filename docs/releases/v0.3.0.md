# MiniOS v0.3.0 — "See the Machine"

**发布日期**: 2026-02-26

## 版本概览

v0.3.0 让 MiniOS 从"能教人 OS"升级到"能让人看见 OS 内部运转"。新增 `explain` 教学命令、引导式 `tutorial`、中断统计、调度器配置查看、以及大幅扩展的 `/proc` 虚拟文件系统。

## 新增特性

### explain 命令 — 命令级教学
```
MiniOS $ explain ls
=== How 'ls' works ===

1. Shell parses input → command='ls', args=[path]
2. VFS resolves the path by walking the inode tree
3. For each child inode: read type and size, print

Key concepts:
  Inode — a numbered record storing file metadata + data
  VFS   — Virtual File System, abstracts storage backends
```

支持 9 个命令的详细解释: ls, cat, ps, meminfo, trace, spawn, sched, pagetable, frames

### tutorial 命令 — 引导式学习
首次启动提示 `Type 'tutorial' to start learning`，展示 10 步引导流程，覆盖进程、内存、文件系统、trace 四大主题。

### interrupts 命令 — 中断统计
```
MiniOS $ interrupts
IRQ  NAME       COUNT     RATE
0    Timer      15234     ~100/s
1    Keyboard   47        on-demand
```

### sched config — 调度器配置
```
MiniOS $ sched config
MLFQ Configuration:
  Queue 0 [HIGH]: time_slice = 2 ticks
  Queue 1 [MED]:  time_slice = 4 ticks
  ...
  Total switches: 150
```

### /proc 虚拟文件系统扩展
新增 4 个 procfs 条目:
- `/proc/interrupts` — 中断计数
- `/proc/scheduler` — 调度器运行状态
- `/proc/version` — OS 版本信息
- `/proc/trace` — trace 缓冲区统计

### 启动优化
移除 boot-time 冗余输出 (syscall/IPC smoke test, tick 日志)，减少引导噪音。

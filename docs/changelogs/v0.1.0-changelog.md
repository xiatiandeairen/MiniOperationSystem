# v0.1.0 Changelog — 全部开发变更

## 核心功能
- `feat(common): enable cross-crate type sharing and trait contracts` — 10 traits, 6 error types, 15+ 共享类型
- `feat(hal): implement complete HAL crate` — serial UART, VGA text, GDT/TSS, IDT (6 handlers), PIC 8259, keyboard
- `feat(hal): render text on pixel framebuffer for visible QEMU output` — 8×16 bitmap font, 160×45 字符, BGR 颜色
- `feat(trace): implement Trace Engine` — 4096-slot Ring Buffer, SpanGuard RAII, JSON export
- `feat(memory): implement memory subsystem` — Bitmap 帧分配器, 4级页表, 1MiB 链表堆
- `feat(process,scheduler): implement process management + MLFQ scheduler` — PCB, global_asm 上下文切换, 4级 MLFQ
- `feat(fs): implement filesystem subsystem` — VFS + RamFS (inode-based) + ProcFS
- `feat(ipc): implement IPC message queues` — 16 队列, 256字节载荷
- `feat(syscall): implement syscall dispatcher` — 7 个 syscall (read/write/getpid/yield/exit/uptime/meminfo)
- `feat(shell): implement interactive terminal` — 15 命令, 行编辑, 键盘输入
- `feat(kernel): boot MiniOS in QEMU with serial and VGA output` — bootloader_api v0.11 入口

## Bug 修复
